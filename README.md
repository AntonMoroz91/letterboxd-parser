# Парсер оценок Letterboxd

## Постановка задачи
Разработать инструмент для сбора всех оценок пользователя Letterboxd. Парсер принимает логин пользователя, парсит постраничный дневник (Diary) и извлекает:
- название фильма
- год выпуска
- оценку пользователя (от 0.5 до 10)

Результаты сохраняются в CSV и Excel.

## Инструкция по установке и запуску

### Требования
- Python 3.7 или выше

### Установка
1. Клонируйте репозиторий:
   git clone https://github.com/AntonMoroz91/letterboxd-parser.git
   cd letterboxd-parser
2. Установите зависимости:
   pip install -r requirements.txt

### Запуск
python letterboxd_parser.py [логин]

Если логин не указан, по умолчанию используется rfeldman9.

### Результат
Программа создаёт два файла:
- [логин]_rates.csv
- [логин]_rates.xlsx
