# Парсер оценок Letterboxd

## Постановка задачи
Разработать инструмент для сбора всех оценок пользователя Letterboxd. Парсер принимает логин пользователя и возвращает структурированные данные (название фильма, год выпуска, оценку) с возможностью сохранения в CSV или Excel.

## Инструкция по установке и запуску

### Требования
- Python 3.7 или выше
- Установленные библиотеки: pandas, openpyxl

### Установка
1. Клонируйте репозиторий:
   git clone https://github.com/ваш-username/letterboxd-parser.git
   cd letterboxd-parser
2. Установите зависимости:
   pip install -r requirements.txt

### Запуск
   python letterboxd_parser.py

### Результат
Скрипт создаст два файла:
- `[логин]_rates.csv` — данные в формате CSV
- `[логин]_rates.xlsx` — данные в формате Excel
